# Ionic-aluracar
Aprendendo ionic com o curso da Alura

## Como rodar

### Inicie a API
Em um terminal rode:
- cd aluracar-webservice/
- npm start

### Inicie o projeto ionic
Em outro terminal rode:
- cd Ionic-aluracar/
- ionic serve --lab
